# SSDC Animated Website

A premium single-page Next.js landing page with:

- custom cinematic preloader
- animated custom cursor
- floating particle background
- parallax and scroll animations
- mobile menu
- SSDC-oriented institutional content
- styled contact form ready for backend integration

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Production-ready next steps

You can wire the contact form to any of these:

- Next.js API route
- Formspree
- Resend
- EmailJS
- custom backend endpoint

You can also replace the placeholder SSDC copy and contact email with your exact organizational content.
